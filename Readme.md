# Koco

Koco is an experimental GPU programming language written in Rust.

The language is inspired by Rust and aims to provide a more expressive alternative to existing shader languages while remaining focused on GPU programming. Koco currently supports features such as enums and pattern matching, enabling higher-level abstractions that are uncommon in modern shader languages.

## Features

* Rust-inspired syntax
* Enums (algebraic data types)
* Pattern matching
* Static type checking
* GPU-focused design

## Goals

* Compile-time generics
* Traits
* Modern tooling and diagnostics
* Efficient code generation
* Integration with the Rust ecosystem

## Status

Very Early development and subject to significant change.
